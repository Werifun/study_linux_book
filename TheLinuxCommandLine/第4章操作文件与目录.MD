- in: 创建硬链接和符号链接
  
  - 创建硬链接：ln file link (把file链接到link上)
    | 硬链接 | 名称部分 | 数据部分 |
    | :----: | :------: | :------: |
    |        | 文件名称 | 文件内容 |
  
    - 硬链接其实就是创建额外的**名称文件**，类似于别名。
    
  - 创建符号链接：ln -s file link(个人理解-s是soft的意思)
  
  - 硬链接的缺点：
  
    - 无法跨磁盘引用（不能引用自身文件系统以外的文件）
  - 无法引用目录（文件夹）
    
  - 软连接的注意点：
  
    - 删掉源文件的时候，如果软链接没有删除，会导致**坏链接**
